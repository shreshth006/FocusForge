# FocusForge - Smart Study Planner

FocusForge is a JavaScript-based productivity web app built for BCSE203E (Web Programming) Digital Assignment II.

## Live Features
- Add study tasks with title, priority, and due date
- Mark tasks complete and delete tasks
- Clear all completed tasks
- Focus timer with 3 modes: Pomodoro (25), Short Break (15), Deep Work (45)
- Track daily summary: total tasks, completed tasks, and focus sessions
- Persistent data using Local Storage

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage

## How to Run Locally
1. Download or clone this repository.
2. Open `index.html` in any modern browser.

## GitHub Pages Deployment
1. Push files to your GitHub repository.
2. Go to Settings > Pages.
3. Select branch: `main` and folder: `/root`.
4. Save and open the generated GitHub Pages URL.
