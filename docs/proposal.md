# Project Proposal

## Project Title
FocusForge - Smart Study Planner with Focus Timer

## Objective
The objective of FocusForge is to help students manage study tasks and maintain concentration using a timer-based workflow. The app solves the common problem of poor time management and low productivity during self-study.

## Key Features
1. Task creation with title, due date, and priority
2. Task completion tracking and deletion
3. Focus timer with multiple modes (Pomodoro, Short Break, Deep Work)
4. Daily summary dashboard showing task and focus statistics
5. Data persistence through Local Storage

## Technologies
- JavaScript (Vanilla)
- HTML5
- CSS3
- Browser Local Storage API
- GitHub for version control and hosting

## Why This Project Is Suitable
- Practical for students
- Demonstrates DOM manipulation, events, data handling, and timers
- Has a clear user interface and measurable output
- Deployable on GitHub Pages for easy evaluation
